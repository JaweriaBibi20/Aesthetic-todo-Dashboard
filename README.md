# ✨ ZenTask | Premium Productivity Board

ZenTask is a minimalist, high-class personal task management interface designed with a clean, modern soft-light theme. It replicates the premium feel of modern productivity tools like Notion and Asana, utilizing pure frontend styling techniques.

## 🛠️ Tech Stack
* **HTML5:** Semantic architecture for structured layouts.
* **CSS3:** Custom variables (CSS variables), Flexbox, CSS Grid, and dynamic state transitions.

## ✨ Key Features
* **Pure CSS Interactivity:** Task completion logic (checkbox tracking with text strike-through) is handled completely via CSS without a single line of JavaScript.
* **Premium Soft Aesthetic:** Ditch the heavy dark themes for a clean, eye-soothing white palette accented with soft lavender, peach, and mint green colors.
* **Organized Categorization:** Built-in pill tags to visually sort tasks by tech domains (Database, Web Dev, Java, Assembly).
* **Fully Responsive:** Adapts beautifully across different screen dimensions.

## 📂 Project Structure
* `task.html` - The main application workspace structure.
* `task-style.css` - Custom styling stylesheet managing colors, grids, and toggle logic.
